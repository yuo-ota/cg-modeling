## 作品概要
本作品は、懸垂線*の軌道を物理演算によらず数式により求められる特性を活かし、ユーザーが自身でその軌道を生成・体験できることを目的としています。

\*懸垂線：物体を2点で吊るした際に形成される曲線であり、以下のような式で表されることが知られています。
$$y = a \cos\left(\frac{x - b}{a}\right)$$
